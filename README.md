# 🤖 RagMaster AI

RagMaster AI is a Retrieval-Augmented Generation (RAG) application that allows users to chat with PDFs and websites using Google Gemini, LangChain, FAISS, and Hugging Face Embeddings.

## 🚀 Live Demo

https://ragmaster-ai.streamlit.app

## 🎥 Demo Video

https://github.com/user-attachments/assets/fd344074-55eb-4e67-9d5a-39281d3c4b14

## ✨ Features

* Chat with PDF documents
* Chat with website content
* AI-powered answers using Gemini
* Semantic search with FAISS
* Fast and simple Streamlit interface

## 🛠️ Tech Stack

* Python
* Streamlit
* LangChain
* Google Gemini
* FAISS
* Hugging Face Embeddings
* PyPDF
* BeautifulSoup

## ⚙️ How It Works

```text
PDF / Website
     ↓
Document Loader
     ↓
Text Chunking
     ↓
Embeddings
     ↓
FAISS Retrieval
     ↓
Gemini AI
     ↓
Answer
```

## 📦 Run Locally

```bash
git clone https://github.com/Abhilash-bheri/AI-RAG-Assistant.git
cd AI-RAG-Assistant

pip install -r requirements.txt

streamlit run app.py
```
